# 随机点名应用

处理 Excel 用的是 [js-xlsx](https://github.com/SheetJS/js-xlsx)

## 操作
1. 用浏览器打开 index.html
2. 输入 Excel 格式的名单文件
3. 点击开始按钮，名字开始在视窗中滚动显示
4. 点击结束按钮，有一个名字停留在视窗中，即得到随机名字

## 输入文件说明
1. 输入的 Excel 文件中只能有一个 Sheet
2. 输入的 Excel 文件中必须包含姓名列，该列表头在第一行，表头内容为 “姓名”
